The query string parameters are
  • s=the%20message
  • stroke-width=3
  • bg-stroke=black
  • bg-fill=white
  • fg-stroke=red
  • fg-fill=transparent
  • variance=0.25
  • shadow=200
  • shadow-x=200
  • shadow-y=200
  • pixel-size=200
  • line-offsets=-1,2,0,1 (sometimes necessary for adjusting things when you have multiple lines and letters with ascenders/descenders)
